This project purpose is to authentically reproduce the Publication of the Hebrew Bible (1866) as was done by Meir (Max) Letteris
ספר תורה, נביאים וכתובים : מדויק היטב על פי המסורה 
Sefer Torah Neviʼim u-Khetuvim : meduyaḳ heṭev ʻal pi ha-Masorah
Author:	הוגה בעיון נמרץ על ידי מאיר הלוי לעטעריס. לעטעריס.  Meir Letteris

Please edit only the individual books. The full text is overwritten by the individual books.

Letteris comments are in a seperate file Comments_utf8.txt

Editing comments are in separate files for individual books (might move to wiki pages as needed) eg: Genesis_editing.txt

Resources used:
Three scans of different prints of the Letteris Bible.


Tools used (recommended)
Github desktop: to download and update with the main data
BabelPad: to edit UTF8 files easily
BabelMap: a tool to see character mappings (not necessary)
EzraSIL251: Font with good placing of Nikudot and Taamim
